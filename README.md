# Tracefile Academy

Tracefile Academy is an introductory forensic science learning platform built using TypeScript and Vite.  
The project is designed for educational purposes and demonstrates frontend development concepts, interactive modules, quizzes, and progress tracking.

## Features
- Interactive forensic science learning modules
- Mock case studies
- Quiz-based knowledge checks
- Progress tracking
- Certificate of completion (non-accredited)

## Technologies Used
- TypeScript
- Vite
- HTML
- CSS

## How to Run This Project Locally
1. Clone the repository
2. Open the project folder in VS Code
3. Run `npm install`
4. Run `npx vite`
5. Open the local server URL provided in the terminal

## Disclaimer
This project is for educational and demonstration purposes only.  
It does not provide professional or accredited forensic certification.
